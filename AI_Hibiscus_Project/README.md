# AI-Powered Hibiscus Leaf Disease Classification

This project focuses on detecting and classifying diseases in hibiscus plant leaves using machine learning techniques.

## Project Overview
The aim of this project is to build a model that can identify leaf diseases based on image features, helping in early detection and better plant health management.

## Key Features
- Image-based disease detection
- Data preprocessing and feature extraction
- Model training and evaluation

## Tools & Technologies
- Python
- Machine Learning
- Image Processing
- Jupyter Notebook

## Outcome
This project helps in understanding how AI can be applied in agriculture for plant disease detection.
